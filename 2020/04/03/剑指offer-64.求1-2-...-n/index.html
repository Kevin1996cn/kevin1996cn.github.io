剑指offer-64.求1+2+...+n

Apr 3, 2020 #题解 #剑指offer #回溯

一些C++ STL的使用，和一些刷题需要注意的问题，例如数据范围、初始化、全排列、报错等。

求 1+2+…+n ，要求不能使用乘除法、for、while、if、else、switch、case等关键字及条件判断语句（A?B:C）。

示例 1：

输入: n = 3 输出: 6 示例 2：

输入: n = 9 输出: 45

分析：这些都不让用就只能递归了。一开始居然脑子抽了没想到。太蠢了><

class Solution {
public:
    int sumNums(int n) {
        if(n==1)
        return 1;
        else
        return n+sumNums(n-1);
    }
};
